# 재산세 계산기 다가구주택 계산 방법 개선 PRD

## 프로젝트 개요
다가구주택 재산세 계산 시 민원인 설명란의 '3. 재산세 본세 계산' 부분에서 각 구별 과세표준에 해당하는 세율을 정확히 적용하여 표시하는 기능을 개선합니다.

## 현재 문제점
- 다가구주택 선택 시 민원인 설명란의 '3. 재산세 본세 계산' 부분에 계산 과정이 제대로 표시되지 않음
- 전체 과세표준에 세율을 적용하는 방식으로 표시될 가능성이 있음

## 요구사항

### 기능 요구사항
1. 다가구주택 선택 시 각 구별 과세표준에 해당하는 각각의 세율을 적용
2. 민원인 설명란의 '3. 재산세 본세 계산' 부분에 상세한 계산 과정 표시
3. 1세대 1주택 특례세율 적용 시 정확한 계산 공식 표시
4. 각 구별 계산 결과와 최종 합계 과정 명시

### 예시 요구사항
- 주택유형: 다가구
- 주택공시가격: 568,000,000원
- 1구 과세표준: 94,205,773원 (특례세율 적용)
- 2구 과세표준: 75,924,332원 (특례세율 적용)  
- 3구 과세표준: 75,924,332원 (특례세율 적용)
- 최종 결과: 재산세 본세 159,650원

### 기술 요구사항
- React + TypeScript 환경
- 기존 디자인 유지
- 계산 로직의 정확성 보장
- 사용자 친화적인 설명 제공

## 성공 기준
1. 다가구주택 선택 시 민원인 설명란에 각 구별 계산 과정이 정확히 표시됨
2. 예시 데이터 입력 시 예상 결과와 일치하는 계산 결과 출력
3. 기존 일반 주택 계산 기능에 영향 없음
